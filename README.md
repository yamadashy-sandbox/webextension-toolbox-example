# webextension toolbox example

webextension-toolbox のサンプルプロジェクトです。

Qiita記事で解説しています。

![](./promo/Screenshot_1280x800.png)

# Development
- Node.js v10.16.0

## Setup
```
$ yarn
```

## Watch and build
```
yarn dev chrome
yarn dev firefox
yarn dev opera
yarn dev edge
```

## Build extension
```
yarn build chrome
yarn build firefox
yarn build opera
yarn build edge
```
